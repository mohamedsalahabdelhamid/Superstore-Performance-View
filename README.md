🛒 Superstore Sales Analytics Dashboard
End-to-End Data Analysis Project (Excel | Power Query | Power Pivot | DAX)
📌 Project Summary

This project demonstrates a complete end-to-end data analytics workflow using Microsoft Excel as a Business Intelligence tool.

The objective was to transform raw retail sales data into a structured analytical model and build an interactive executive dashboard to support strategic decision-making.

The project covers:

Data Cleaning ➜ Data Transformation ➜ Data Modeling ➜ DAX Calculations ➜ Business Insights ➜ Dashboard Design

🧠 Business Problem

The company needed visibility into:

Revenue and Profit performance

Top-performing products and customers

Regional and segment-based sales distribution

Shipping efficiency impact on sales

Category and sub-category contribution

The goal was to centralize KPIs and performance metrics into one interactive reporting solution.

🛠 Technical Implementation
🔹 1. Data Preparation – Power Query (ETL)

Cleaned raw transactional dataset

Removed duplicates & handled missing values

Standardized data types and columns

Built structured tables ready for modeling

Optimized data structure for performance

🔹 2. Data Modeling – Power Pivot

Built relational data model

Applied Star Schema Design

Created relationships between fact and dimension tables

Ensured scalable and optimized model structure

🔹 3. DAX Measures Developed

Total Sales

Total Profit

Sales by Category

Sales by Region

Top 10 Dynamic Ranking

Shipping Status Distribution

Segment Contribution

📈 Key Business Insights

Technology category generated the highest revenue contribution

Certain regions showed high sales but lower profitability

A small percentage of customers generated a significant portion of total revenue (Pareto effect)

Shipping delays correlated with lower customer segment performance
